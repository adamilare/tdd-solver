# 📖 TDD SOLVER <a name="about-project"></a>

**TDD Solver** is a ruby project to demonstrate the concept of Test-Driven Development. The test is written before writing implementation details that work towards making all defined tests to pass.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Ruby

### Key Features <a name="key-features"></a>

- Tests before development
- Implementation that passes all defined tests

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: Ruby installed on your computer.

### Setup

Clone this repository to your desired folder:
``

- `git Clone https://github.com/adamilare/tdd-solver.git` to your desired folder.
- `cd tdd-solver`
- run `bundle install`

### Usage

To run the project, execute the following command with your terminal opened to the project folder:

    rspec --format doc spec
